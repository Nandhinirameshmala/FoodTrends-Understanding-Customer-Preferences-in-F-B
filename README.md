## 📌 Project Overview
FoodTrends is a data analysis and visualization project built using Microsoft Power BI to explore dietary habits, nutritional intake, and customer food preferences.

The dashboard provides insights into how food choices vary based on health conditions, lifestyle, and demographics, helping identify meaningful trends in nutrition and eating behavior.

---

## 🎯 Objective
- Analyze food consumption patterns  
- Compare nutritional values (calories & protein)  
- Understand dietary preferences  
- Identify the impact of health conditions on food choices
