FoodTrends – Understanding Customer Preferences in F&B
Project Overview

FoodTrends is a data visualization project developed using Microsoft Power BI to analyze dietary habits, nutritional intake, and food preferences. The dashboard provides insights into how different foods contribute to nutritional values such as calories, protein, and fat while exploring dietary patterns across different groups.

The project aims to help understand food trends, nutritional distribution, and lifestyle impacts through interactive visualizations.

Objectives
Analyze customer dietary preferences.
Compare nutritional values across different food items.
Understand the relationship between lifestyle habits and nutrition.
Visualize dietary patterns and health-related insights.

Dashboard Sections
1. Dietary Preferences

This section analyzes different diet types followed by individuals, including:

Omnivore
Pescatarian
Vegetarian
Vegan

It helps understand how dietary choices are distributed among participants.

![Dietary Preferences and Averages](dashboard.png)

2. Nutrients and Disease

This section compares nutritional values such as calories and protein across different food items while allowing analysis based on health conditions such as:

Diabetes
Hypertension
Healthy individuals

This helps identify how diet and nutrition relate to different health conditions.

![Nutrition and Disease Analysis](dashboard.png)

3. Averages by Breakfast Suggestions

This section analyzes average nutritional values from breakfast food suggestions. It helps identify which foods provide better nutritional balance for starting the day.

![Averages By Breakfast Suggestions](dashboard.png)

4. Lifestyle & Nutrition Impact

This section explores the relationship between lifestyle activity levels and nutritional intake. Activity levels include:

Sedentary
Moderately Active
Very Active
Lightly Active
Extremely Active

This helps understand how lifestyle habits influence nutritional needs.

![Lifestyle & Nutrition Impact](dashboard.png)

Key Metrics
Total Fat
Total Protein
Average Calories
Average Rating

These metrics provide an overall summary of the nutritional data analyzed in the dashboard.

Data Source

The dataset used for this project was collected from Kaggle, a popular platform for data science datasets and machine learning projects. The data includes information about food items, nutritional values, dietary preferences, health conditions, and lifestyle activity levels.

This dataset was used to analyze nutritional patterns, dietary habits, and their relationship with lifestyle and health conditions.
